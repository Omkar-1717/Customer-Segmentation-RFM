# 📊 Customer Segmentation using RFM Analysis

## 📌 Project Overview
This project applies **RFM (Recency, Frequency, Monetary) analysis** to transactional customer data to identify segments such as **Loyal Customers**, **At Risk Customers**, and **New Customers**.

## ⚙️ Workflow
1. Data Cleaning & Preparation  
2. RFM Metric Calculation  
3. Scoring & Segmentation  
4. Visualization (Seaborn + Plotly)  
5. Insights & Marketing Recommendations  

## 📂 Files
- `customer_data.csv` → Raw transactional dataset  
- `RFM_Project.ipynb` → Jupyter Notebook with analysis & dashboard  
- `README.md` → Project documentation  

## 📊 Visualizations
- Segment distribution (Pie Chart)  
- Average spend by segment (Bar Chart)  
- Recency vs Frequency scatter plot  
- Frequency vs Monetary bubble chart  

## 🎯 Insights
- **Loyal Customers** → Reward programs, VIP offers  
- **At Risk Customers** → Win‑back campaigns, personalized discounts  
- **New Customers** → Onboarding offers, bundle deals  

## 🏆 Conclusion
RFM segmentation enables businesses to personalize marketing, improve retention, and maximize profitability through data‑driven decisions.
